01 AI Vibe Coding Tools 🤖
Cursor cursor.sh ↗

The gold standard for AI-first coding. Full VS Code fork with deeply integrated Claude & GPT-4o. Best for full-stack projects.

Freemium VS Code fork Chat + Compose
Windsurf codeium.com ↗

By Codeium. Agentic AI editor that can autonomously write, run, and debug code. Very generous free tier.

Free Tier Agentic VS Code fork
Bolt.new bolt.new ↗

Build full-stack apps from a prompt in the browser. Instant preview, deploys to Netlify. Best for rapid prototyping.

Freemium Browser-based Full-stack
v0 by Vercel v0.dev ↗

Generate React + Tailwind UI components from prompts. Integrates with Next.js. Incredible for frontend & landing pages.

Freemium React/Next.js UI focus
Lovable lovable.dev ↗

Prompt-to-app with Supabase integration baked in. Gets you from idea to a deployed app with auth + DB very fast.

Freemium Full-stack Supabase
Claude Code docs.claude.com ↗

Anthropic's agentic CLI tool. Reads your whole codebase, writes, runs, tests code. Best for complex multi-file work.

Freemium CLI agent Codebase-aware
Startup Tip: Start with Bolt.new or Lovable for your MVP prototype. Move to Cursor or Windsurf once you need real version control and custom logic.
02 Database & Storage 🗄️
Supabase supabase.com ↗

Open-source Firebase alternative. PostgreSQL + Auth + Storage + Realtime + Edge Functions. Best all-in-one for startups.

Free Tier Open Source PostgreSQL
PlanetScale planetscale.com ↗

Serverless MySQL with branching like Git. Scales to massive traffic. Great developer experience.

Freemium MySQL Serverless
Neon neon.tech ↗

Serverless PostgreSQL with autoscaling and branching. Scales to zero when idle — perfect for early-stage apps.

Free Tier PostgreSQL Serverless
MongoDB Atlas mongodb.com ↗

Managed NoSQL document database. 512MB free forever. Great for flexible data structures and rapid iteration.

Free Tier NoSQL Document DB
Upstash upstash.com ↗

Serverless Redis & Kafka. Perfect for caching, rate limiting, sessions, and queues. Free up to 10K requests/day.

Free Tier Redis Caching
Cloudflare R2 cloudflare.com ↗

S3-compatible object storage with zero egress fees. 10GB free forever. Ideal for images, files, and backups.

Free Tier Object Storage S3 compat
Startup Tip: Use Supabase as your default — it covers database, auth, file storage, and realtime in one platform with a great free tier.
03 Backend, API & Server 🔌
Hono hono.dev ↗

Ultrafast web framework for edge runtimes (Cloudflare Workers, Deno, Bun). Minimal, typed, and production-ready.

Free Open Source Edge-first
FastAPI fastapi.tiangolo.com ↗

Python API framework with automatic OpenAPI docs. Best if you're building ML/AI backends. Typed, fast, and beloved.

Free Open Source Python
tRPC trpc.io ↗

End-to-end typesafe APIs without schemas or code gen. If your stack is TypeScript, this eliminates a ton of boilerplate.

Free Open Source TypeScript
Cloudflare Workers workers.cloudflare.com ↗

Serverless edge compute. 100K requests/day free. Deploy JS/TS functions globally with zero cold starts.

Free Tier Edge Serverless
Railway railway.app ↗

Deploy any server (Node, Python, Go, Docker) in seconds. Persistent containers with $5/mo free credit. GitHub auto-deploy.

Freemium Any language Docker
Inngest inngest.com ↗

Event-driven background jobs and workflows. Handle async tasks, queues, cron jobs with retries. Free tier available.

Free Tier Background Jobs Queues
Startup Tip: For a JS/TS startup, use Next.js API routes + tRPC to start. Add Cloudflare Workers for high-traffic edge endpoints. For Python/AI workloads, use FastAPI on Railway.
04 Authentication & Auth 🔑
Clerk clerk.com ↗

Drop-in auth with beautiful pre-built UI components. Social login, MFA, organizations. Free up to 10,000 MAU.

Free Tier React/Next.js Pre-built UI
Auth.js (NextAuth) authjs.dev ↗

The open-source auth solution for Next.js. Supports 50+ OAuth providers. No vendor lock-in, self-hostable.

Free Open Source Next.js
Supabase Auth supabase.com ↗

Built-in auth in Supabase. Email, social, magic links, phone OTP. If you're already on Supabase, use this.

Free Tier Open Source Integrated
Better Auth better-auth.com ↗

New framework-agnostic auth library for TypeScript. Comprehensive, self-hosted, and free. Growing quickly.

Free Open Source TypeScript
05 Web Security 🔒
Cloudflare cloudflare.com ↗

Free WAF, DDoS protection, CDN, bot management, and SSL. Every startup should use Cloudflare in front of their app.

Free Tier WAF DDoS CDN + SSL
Snyk snyk.io ↗

Scan your code and dependencies for vulnerabilities. Free tier for open source and small teams. Integrates with GitHub.

Free Tier Vuln Scanning CI/CD
OWASP ZAP zaproxy.org ↗

Free open-source web app security scanner. Find XSS, SQLi, CSRF vulnerabilities automatically. Industry standard.

Free Open Source Pen Testing
Arcjet arcjet.com ↗

Developer-first security SDK for Next.js/Node. Rate limiting, bot protection, email validation in a few lines of code.

Free Tier Rate Limiting Bot Protection
Let's Encrypt letsencrypt.org ↗

Free, automated SSL/TLS certificates. Most hosting platforms (Vercel, Railway, etc.) handle this automatically for you.

Free Open Source SSL/TLS
Doppler doppler.com ↗

Secrets management for teams. Never put API keys in .env files again. Syncs secrets across all environments. Free for individuals.

Free Tier Secrets Mgmt Env Vars
Security Checklist: Always enable Cloudflare (free WAF + DDoS), use HTTPS everywhere, never hardcode secrets (use Doppler), run Snyk in CI, and validate all user inputs. Follow OWASP Top 10.

Free Tier CI/CD Automation
